Freelancer Endpoints

MethodEndpointAuthDescription
POST /api/freelancers Public Register new freelancer (first_name, last_name, email, contact, password, gender, dob)
POST /api/freelancers/verify-email Public Verify email with 6-character verification code
POST /api/freelancers/resend-verification Public Resend email verification code (expires in 30 min)
POST /api/freelancers/login Public Login with email & password, returns Sanctum token
GET /api/freelancers Public List freelancers with search (name/email/contact) & pagination
GET /api/freelancers/{freelancer} Public Get freelancer with skills, work experiences, and shifts
PUT /api/freelancers/{freelancer} Sanctum Update profile (info, skills, work experiences, shifts, image, documents)
DELETE /api/freelancers/{freelancer} Sanctum Delete freelancer account
POST /api/freelancers/logout Sanctum Logout and revoke current token
DELETE /api/freelancers/{freelancer}/work-experiences/{experience} Sanctum Delete a specific work experience entry
DELETE /api/freelancers/{freelancer}/skills/{skill} Sanctum Detach a skill from freelancer profile

Password Reset Endpoints

MethodEndpointAuthDescription
POST /api/forgot-password Public Send password reset email with token link
POST /api/reset-password Public Reset password using token (expires in 1 hour)

MethodEndpointAuthDescription

GET /api/jobs/{id} Public Get single job details with employer
POST /api/jobs Sanctum Create job posting (title, description, skills, rate_type, budget, deadline, shift_type)
PUT /api/jobs/{id} Sanctum Update job (owner employer only)
DELETE /api/jobs/{id} Sanctum Delete job (owner employer only)
